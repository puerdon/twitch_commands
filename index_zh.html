漢的頻道指令

  • Eng
  • 中文

點歌

  • !sl : 歌單連結
  • !sr {歌名} : 點一首歌單中的歌
  • !ll {歌名} {樂譜連結 (選填)} : (現場視譜學) 點一首不在歌單裡的歌
  • !currentSong : 現在在彈的歌曲資訊

聲音指令

  • !clap : 一人掌聲
  • !claps : 多人掌聲
  • !rain : 召喚雨
  • !wind : 召喚風
  • !thunder : 召喚打雷
  • !tear : 滴一滴眼淚
  • !chirp : 召喚鳥叫聲

實用連結

  • !discord : 歡迎加入 :)
  • !tip, !donate : 歡迎用小費丟我、支持我 :)

管理員專區

  • !so {@username} : shoutout to a great streamer. The @username can be with or without the @ symbol. If the streamer uses localized display name like Chinese or Japanese characters, this command still works if they have already appeared in the chat.
  • !srs on|off : turn on/off song request.

其他

  • !lurk , !unlurk
    • : 潛水 / 浮出水面
  • !raid , !subraid : 秀出揪團訊息
  • !hug {@username} : 抱抱指令，如果你想一次抱多人，請用空格隔出你要抱的使用者帳號， 例如：!hug {@han_bot} {@han__bot} {@han___bot}
  • !love : 秀出15個隨機愛心
  • !uptime : 看我直播多久了
  • !taiwantime : 台灣報時系統
