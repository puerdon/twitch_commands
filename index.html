Available commands in Han's (padahati) Twitch chat

  • Eng
  • 中文

Song Request

  • !sl : get songlist
  • !sr {song title} : request a song from the songlist
  • !ll {song title} {sheet url (optional)} : (live learning) request a song NOT from the songlist
  • !currentSong

Sound alert

  • !clap : one person clapping
  • !claps : many people clapping
  • !rain : summon the rain
  • !wind : summon the wind
  • !chirp : birds chirpping

Useful links

  • !discord : Welcome to join :)
  • !tip, !donate : Welcome to support me :)

MOD only

  • !so {@username} : shoutout to a great streamer. The @username can be with or without the @ symbol. If the streamer uses localized display name like Chinese or Japanese characters, this command still works if they have already appeared in the chat.
  • !srs on|off : turn on/off song request.

Others

  • !lurk , !unlurk
  • !raid , !subraid: show raid message
  • !hug {@username} : hug somebody. If you want to hug many people at the same time, please use space to separate the user IDs: !hug {@han_bot} {@han__bot} {@han___bot}
  • !love : show 15 random heart emotes
  • !uptime : show stream duration
  • !taiwantime : show current time in Taiwan
